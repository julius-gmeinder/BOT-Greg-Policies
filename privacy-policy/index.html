Privacy Policy

Last Updated: 08.06.2025

This Privacy Policy describes how BOT Greg ("the Bot") collects, uses, and shares information when users interact with it on Discord.

  1. Information Collected

    2. The Bot may collect and store the following information:

    • Discord user IDs
    • Server IDs and channel IDs where the Bot is used
    • Usernames and discriminators
    • Content of commands/messages specifically directed to the Bot
    • Usage data (command frequency, etc.)

  2. How Information is Used

    3. Collected information is used to:

    • Provide and maintain the Bot's functionality
    • Improve user experience
    • Troubleshoot technical issues
    • Enforce terms of service
    • Generate aggregated, non-personal statistics

  3. Data Storage

    4. User data is stored securely and is only retained as long as necessary to provide services. Users may request data deletion by contacting the developer.

  4. Third-Party Sharing

    5. User data will not be sold or shared with third parties except:

    • When required by law
    • With Discord as necessary for Bot operation
    • With service providers that assist in Bot operation (hosting, analytics)
    • With partners or sponsors that support the Bots development financially

  5. User Rights

    6. Users may:

    • Request access to their stored data
    • Request correction of inaccurate data
    • Request deletion of their data (may limit Bot functionality)

  6. Changes to This Policy

    7. The developer reserves the right to modify this policy. Users will be notified of significant changes via the Bot's support server or Discord announcement.
